## 📖 Deskripsi Aplikasi

**Website Wisata Jukut Etam** adalah platform informasi digital yang menyajikan informasi lengkap seputar wisata Jukut Etam — destinasi rekreasi keluarga yang memadukan kolam renang dan pemancingan di kota Samarinda, Kalimantan Timur.

Website ini dibangun dengan tujuan untuk:
- Memperkenalkan Jukut Etam kepada khalayak yang lebih luas
- Menyajikan informasi fasilitas, harga tiket, dan layanan secara transparan
- Menyediakan fitur pemesanan tiket online dengan QR Code digital
- Menjadi media promosi digital untuk mendukung perkembangan wisata lokal

---

## ✨ Fitur Website

### 👤 Halaman Publik (User & Guest)
| Fitur | Deskripsi |
|---|---|
| 🏠 **Beranda** | Halaman utama dengan hero section, fitur unggulan, dan galeri preview |
| 📋 **Fasilitas & Harga** | Informasi lengkap harga tiket kolam renang, pemancingan, dan sewa fasilitas |
| 🎫 **Pemesanan Tiket** | Form pemesanan tiket online dengan QR Code digital (harus login) |
| 📸 **Galeri** | Kumpulan foto-foto fasilitas dan suasana Jukut Etam |
| 📝 **Tentang** | Informasi sejarah, visi misi, dan lokasi Jukut Etam |
| ⭐ **Ulasan** | Pengunjung dapat mengirimkan ulasan dan rating bintang |
| 👤 **Profil User** | Riwayat ulasan, foto yang diunggah, dan riwayat pemesanan |
| 🔐 **Login/Register** | Sistem autentikasi untuk akses fitur user |

### 🔐 Panel Admin (CMS)
| Fitur | Deskripsi |
|---|---|
| 📊 **Dashboard** | Ringkasan statistik galeri, ulasan, dan user |
| 🖼️ **Kelola Galeri** | CRUD foto galeri + approve/reject foto dari user |
| ⭐ **Kelola Ulasan** | Moderasi ulasan (approve/reject) dari pengunjung |
| 🪑 **Kelola Fasilitas** | CRUD data fasilitas yang ditampilkan di halaman depan |
| 💰 **Kelola Harga** | Update harga tiket kolam renang, pemancingan, dan sewa |
| 🎫 **Kelola Pemesanan** | Lihat semua pemesanan + konfirmasi/pembatalan |
| 🔑 **Autentikasi** | Login aman dengan session + modal konfirmasi logout |
| 📱 **Responsive** | Tampilan admin menyesuaikan desktop & mobile |

---

## 🛠️ Tech Stack

- **Backend:** PHP Native dengan arsitektur **Custom MVC + OOP**
- **Database:** MySQL (PDO)
- **Frontend:** HTML5, Bootstrap 5, CSS3, JavaScript
- **Library:** Vue.js 3, Font Awesome 6
- **Notifikasi:** Custom Flash Message + Progress Bar
- **QR Code:** Google Chart API / QR Server API
- **Font:** Playfair Display + Poppins (Google Fonts)
